# Chess Game

2d шахматы с простой логикой (нет шахов, матов, рокировки и т.д.)

## Предварительный требования

Для реализации игры необходимо использовать движок https://oxygine.org/

1. Движение фигур реализовать по клику
    - Первый клик - выбор фигуры.
    - Второй клик - конечное положение фигуры (фигуры передвигаются с применением Tween).
3. Передвижение фигур по правилам шахмат, с возможностью поедания фигур противника (без логики правил шахмат, т.е. шаха, мата рокировки и т.д.).
4. Поедание фигур происходит с удаление элементов (съеденные фигуры должны плавно исчезать).
5. Логика очередности хода.
6. Программа должна запускаться.
7. Код должен быть залит в Git.

### Примечания
- Выбор картинок и поля на свое усмотрение
- Нельзя использовать сторонни библиотеки, кроме oxygine
- Необходимо собрать приложение под linux 9ubuntu)
- Реализация "минимального функционала", чтобы тестовое задание не заняло много времени. По желпнию можно описать шахматы полностью.

## Сборка и запуск приложения в Docker

### Клонирование репозитория
В папке, где хотите разместить проект, введите команду
```
git clone git@github.com:rnaks1m/chess_game.git
```

### Сборка и запуск
Перейдите в корневую папку проекта и введите команду:
```
docker build -t chess_game .
```
После сборки образа, запустите контейнер. Для этого выполните команды:

Необходимо разрешить доступ для проброса графического интерфейса от Docker к X11 серверу хоста. Для этого введите команду:
```
xhost +local:docker
```
Запустите контейнер с пробросом графического интерфейса на хост:
```
docker run -it --rm -e DISPLAY=$DISPLAY -v /tmp/.X11-unix:/tmp/.X11-unix chess_game
```
Закройте доступ после работы программы:
```
xhost -local:docker
```
